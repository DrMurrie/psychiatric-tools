Psychiatric Tool by Dr Benjamin Murrie

[Amazing ADHD assessment generator 3](adhd3.html)

This has everything you need. An interview template to follow with the key things plus DSM 5 TR criteria. Psychoeducatoin. Titration algorithm. Titration and follow up plan generator. Creates a nice adhd report, certificate, script text for e scripts, creates all the letters they might need. Has prescriber guidance per state. Everything. Perfect for beginners and experienced psychiatrists alike. 

[ADHD Handout Generator](ADHD-Patient-Guide-Generator-30.html)
This is an excellent web app that you can use to provide psychoeducation to your patient, and create a titration schedule for them. It generates a nice handout to print or send to your patients. Plus generates the plain text to put in your eScript. You can choose how much info to add to the hand out. Some patients need more and some need less. It also includes non-pharm options. Most of this has been put into the amove ADHD assessment generator.
